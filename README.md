## Pengenalan

Simple Image Drawer adalah aplikasi sederhana yang memungkinkan pengguna untuk menampilkan semua gambar dari sebuah folder. Aplikasi ini menggunakan Python sebagai bahasa pemrograman utama dan KivyMD untuk membuat antarmuka pengguna yang menarik.

## Fitur

- Menampilkan semua gambar dari sebuah folder.
- Antarmuka pengguna yang sederhana dan intuitif.
- Mendukung berbagai format gambar seperti JPEG, PNG, dan lainnya.

## Memulai

Untuk menjalankan aplikasi ini di mesin lokal Anda, pastikan Anda memiliki hal-hal berikut:

### Requirement

- Python (versi 3.x)
- Pip (untuk instalasi dependensi)

### Teknologi yang Digunakan

- Python
- KivyMD (Material Design Components for Kivy)
- Kivy (Framework untuk membuat aplikasi berbasis NUI)

## Penggunaan

1. Pastikan Python dan Pip sudah terinstal di mesin Anda.
2. Instal dependensi dengan menjalankan perintah berikut di terminal atau command prompt:
   ```
   pip install -r requirements.txt
   ```
3. Jalankan aplikasi dengan perintah:
   ```
   python image_drawer.py
   ```
5. Semua gambar dari folder tersebut akan ditampilkan dalam antarmuka aplikasi.
